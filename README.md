# Snort Log Analiz

Bu proje, Snort loglarını basit şekilde okumak için bir bash scripti içerir.

## Dosyalar

- `logoku.sh` - Log dosyasını okuyan bash script  
- `alert.log` - Örnek Snort alert log dosyası  
- `README.md` - Projenin dokümantasyonu  

## Kullanım

1. `logoku.sh` dosyasının çalıştırılabilir olduğundan emin olun:

chmod +x logoku.sh
